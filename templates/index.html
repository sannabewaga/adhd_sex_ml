Welcome to the ADHD & Sex Prediction Project

This project explores behavioral data to build predictive models for two outcomes:

  • ADHD Diagnosis Prediction using Logistic Regression
  • Sex Classification using Logistic Regression

Due to the high dimensionality of the data (20,000+ features), user input is not collected. Instead, we present a summary of model performance, insights from the data, and ethical considerations.
